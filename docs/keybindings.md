# Shared Keymap — Ghostty / Hyper / Cursor

All three surfaces share the same primary bindings for muscle-memory portability. iTerm2/Warp conventions are the basis.

| Action | Ghostty | Hyper | Cursor integrated terminal |
|---|---|---|---|
| New tab | `Cmd+T` | `Cmd+T` | `Cmd+T` |
| Close tab/split | `Cmd+W` | `Cmd+W` | `Cmd+W` |
| Split right | `Cmd+D` | `Cmd+D` | `Cmd+D` |
| Split down | `Cmd+Shift+D` | `Cmd+Shift+D` | `Cmd+Shift+D` |
| Next split | `Cmd+]` | `Cmd+]` | `Cmd+]` |
| Prev split | `Cmd+[` | `Cmd+[` | `Cmd+[` |
| Zoom split | `Cmd+Shift+Enter` | — | `Cmd+Shift+Enter` |
| Clear scrollback | `Cmd+K` | `Cmd+K` | `Cmd+K` |
| Next tab | `Cmd+Shift+]` | `Cmd+Shift+]` | `Cmd+Shift+]` |
| Prev tab | `Cmd+Shift+[` | `Cmd+Shift+[` | `Cmd+Shift+[` |
| Goto tab N | `Cmd+1..9` | default | `Cmd+1..9` |
| Quick terminal (drop-down) | `` Cmd+` `` | — | — |
| Resize split | `Cmd+Alt+Arrow` | — | — |
| Reload config | `Cmd+Shift+,` | — | — |

## Shell-level shortcuts (all surfaces)

| Key | Action |
|---|---|
| `Ctrl+R` | atuin — searchable shell history |
| `Ctrl+T` | fzf — file picker at cursor |
| `Alt+C` | fzf — cd into fuzzy-picked dir |
| `z <partial>` | zoxide — jump to best-ranked matching dir |
